# Weather API Website

A modern, responsive weather website featuring a clean dark theme design with real-time weather data display, forecasts, and comprehensive API feature showcase.

## Features

- **Hero Section** with animated search functionality and live statistics
- **Real-time Weather Display** with current conditions and detailed metrics
- **7-Day Forecast** with interactive cards
- **Feature Showcase** highlighting API capabilities
- **Data Types Overview** displaying available weather metrics
- **Responsive Design** optimized for all screen sizes
- **Modern Dark Theme** with gradient accents and smooth animations

## Pages & Sections

1. **Navigation** - Fixed header with responsive mobile menu
2. **Hero** - Search functionality with gradient background
3. **Current Weather** - Temperature, humidity, wind, pressure, UV index
4. **Forecast** - 7-day weather forecast grid
5. **Features** - 6 feature cards showcasing API capabilities
6. **Data Types** - Overview of available weather data
7. **CTA** - Call to action for API signup
8. **Footer** - Links and branding

## Technologies

- HTML5 (semantic markup)
- CSS3 (custom properties, Grid, Flexbox, animations)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)

## Getting Started

Open `main.html` in your browser to view the website.

## Design

- Color palette: Dark blue background (#0f172a) with blue accents (#3b82f6)
- Typography: Inter font family
- Responsive breakpoints: 1024px (tablet) and 768px (mobile)
