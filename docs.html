Introduction

The Weather API provides real-time weather data, forecasts, air quality metrics, and severe weather alerts via a simple REST interface. All responses are returned in JSON format.

New to the API? Start by creating a free account to get your API key.

Authentication

Every request to the API must include your API key. You can pass it either as a query parameter or in the Authorization header.

Query parameter

cURL
GET https://api.weather.com/v1/current?city=London&appid=YOUR_API_KEY

Bearer token (recommended)

cURL
GET https://api.weather.com/v1/current?city=London \
  -H "Authorization: Bearer YOUR_API_KEY"

Base URL

All API endpoints are relative to:

https://api.weather.com/v1

Current Weather

Get current weather conditions for any location worldwide.

GET /current

Query Parameters

Parameter Type Required Description
city string Required City name (e.g., London, Tokyo)
units string Optional imperial (Fahrenheit) or metric (Celsius). Default: imperial
lang string Optional Language code for weather descriptions. Default: en

Example Request

cURL
curl -G "https://api.weather.com/v1/current" \
  --data-urlencode "city=London" \
  -d "units=imperial" \
  -H "Authorization: Bearer YOUR_API_KEY"

Example Response

{
  "city": "London",
  "country": "GB",
  "temperature": 62,
  "feels_like": 59,
  "condition": "Partly Cloudy",
  "humidity": 72,
  "wind_speed": 12,
  "wind_direction": "NW",
  "pressure": 1013,
  "visibility": 6.2,
  "uv_index": 4
}

5-Day Forecast

Get a 5-day weather forecast with 3-hour intervals.

GET /forecast

Query Parameters

Parameter Type Required Description
city string Required City name
units string Optional imperial or metric. Default: imperial

Example Request

cURL
curl "https://api.weather.com/v1/forecast?city=London&units=metric" \
  -H "Authorization: Bearer YOUR_API_KEY"

Example Response

{
  "city": "London",
  "forecast": [
    {
      "time": "2026-04-05T12:00:00Z",
      "temperature": 15,
      "feels_like": 13,
      "condition": "Partly Cloudy",
      "precipitation_chance": 30
    }
    // ... more entries
  ]
}

Air Quality

Get current air quality data including AQI level and specific pollutants.

GET /air-quality

Query Parameters

Parameter Type Required Description
city string Required City name
lat, lon number Optional Coordinates instead of city name

Example Response

{
  "aqi": 2,
  "aqi_label": "Fair",
  "pm2_5": 12.4,
  "pm10": 22.1,
  "no2": 18.3,
  "o3": 45.6
}

Weather Alerts

Get active severe weather alerts for a location.

GET /alerts

Parameters

Parameter Type Required Description
lat, lon number Required Coordinates for the alert area
Weather Alerts is available on the Professional plan and above.

Example Response

{
  "alerts": [
    {
      "event": "Severe Thunderstorm Warning",
      "severity": "extreme",
      "description": "Damaging winds up to 70 mph...",
      "start": "2026-04-05T18:00:00Z",
      "end": "2026-04-05T23:00:00Z"
    }
  ]
}

Error Codes

All errors return a JSON object with code and message.

Error Status Description
INVALID_API_KEY 401 The provided API key is not valid
LOCATION_NOT_FOUND 404 The city or coordinates are not recognized
RATE_LIMIT_EXCEEDED 429 Too many requests. Exceeded plan limit
INTERNAL_ERROR 500 An unexpected server error occurred

Rate Limits

Request limits depend on your subscription tier.

Plan Monthly Limit Max Per Minute
Starter 1,000 10
Professional 100,000 100
Business Unlimited 1,000
Rate limit headers (X-RateLimit-Limit, X-RateLimit-Remaining) are included in every response.
